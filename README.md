# MovieCritics
This program is desktop app coded in java using both intellij IDEA ide and netbeans ide.
The main purpose of the app is to list movies and to rate them if you watched or keep listed as to be watched if you didn't watch them.

REQUIREMENTS
---------------------------------------------------------------------
Insert Mysql database connector for java to your projects libraries.
Posters folder must be on src.
Icons folder must be on src.
You must run the sql.sql file in order to initialize the database.
